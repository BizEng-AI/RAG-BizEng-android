  🔑 GITHUB AUTHENTICATION
═══════════════════════════════════════════════════════════════════

When git asks for credentials, use ONE of these:

OPTION A: Personal Access Token (Recommended for now)
────────────────────────────────────────────────────────────────
1. Go to: https://github.com/settings/tokens/new
2. Description: "Android Push Access"
3. Expiration: 30 days (or as needed)
4. Select scopes:
   ✓ repo (all repo access)
   ✓ workflow (if using GitHub Actions)
5. Click "Generate token"
6. COPY THE TOKEN (you'll only see it once!)
7. When git asks for:
   Username: your-github-username
   Password: PASTE_THE_TOKEN_HERE


OPTION B: SSH Key (Better for repeated use)
────────────────────────────────────────────────────────────────
If you already have SSH set up:
  → Just push, it will use your SSH key automatically

If not set up yet:
  1. Generate key: ssh-keygen -t ed25519 -C "your@email.com"
  2. Copy public key: notepad %USERPROFILE%\.ssh\id_ed25519.pub
  3. Add to GitHub: https://github.com/settings/ssh/new
  4. Change remote to SSH:
     git remote set-url origin git@github.com:BizEng-AI/android.git
  5. Push: git push -u origin main

  3. Try again
