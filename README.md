# BizEng Studio

Business English practice app focused on **real-world communication**, not passive learning.

## 🚀 Try the App

Download the latest APK from Releases:

👉 https://github.com/BizEng-AI/RAG-BizEng-android/releases/download/1.0/BizEngChatApp.apk

Install it on your Android device and try it directly.

## What it does

BizEng Studio helps you improve business English through:

- 💬 Chat-based practice with AI
- 🎭 Roleplay scenarios (interviews, meetings, complaints)
- 🎤 Speaking mode with voice input
- 📊 Progress tracking and activity stats
- 🧑‍💼 Admin dashboard for managing users and activity
